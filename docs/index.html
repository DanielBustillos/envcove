Features

Everything you need to manage secrets.

Built for developers who work with multiple projects and providers every day.

Touch ID Protected

App launch and sensitive operations (reveal, export) are gated behind biometric authentication. Re-authentication is required each time.

Keychain Storage

Secret values never touch the filesystem. They live exclusively in the macOS Keychain and are flushed back on every app suspend or quit.

Projects & Providers

Organize secrets into projects and group them by provider. Sidebar navigation lets you filter by provider across all projects at once.

Import & Export

Export to .env or JSON. Import back with full support for commented lines, quoted values, and export KEY=VAL syntax.

Masked by Default

All values are displayed as ••••••••• — fixed-length regardless of actual size, so no length information is ever leaked.

One-click Copy

Hover any secret row to reveal a copy button. Values go straight to the clipboard without ever being displayed on screen.

Security Model

Metadata and secrets never share the same store.

Project names, key names, and provider info are stored as JSON. The actual secret values? Only in the macOS Keychain — ever.
